# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
* ローカルリポジトリ：自分のパソコンの中にある保存場所のこと。
* リモートリポジトリ：インターネット上（GitHubなど）にある共有の保存場所のこと。

## プッシュとマージの違いは何でしょうか？

* プッシュ：ローカルでの作成物をリモートへ送りメンバーで共有できるようにすること。
* マージ：別のブランチの作業内容をブランチに取り込むこと。

## コミットとプッシュの違い

* コミット：ローカルの作成物をリモートへ送り出す前に記録をしておくこと。
* プッシュ：コミットした記録をリモートへ送りだすこと。

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

* 分かりやすく何を変更したか的確に短文で書くこと。

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

* ローカルでマージする方法は、自分自身でチェックをしてそのまま反映することになるので、二重チェックが行われないため慎重に行う必要がある。
* プルリクエストで行うと自分が作成したコードにレビュワーがチェックしてからマージを行うため、ミスや不具合を防ぎやすくなる。

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

* 先にマージされた方を取り込むか、後からマージされた方を取り込むか、どちらの内容も取り込むか状況によって判断する。
